| Boundary | Description |
|----------|-------------|

| `.dgt` file → R reader → C add path | On reload, `.dgt` anchor/landmark coords flow through `read.anchors`/`draw.anchors` → C `anchor_add` (now `marker_add(&g_anchors,…)`). Data-integrity boundary for D-11. |
