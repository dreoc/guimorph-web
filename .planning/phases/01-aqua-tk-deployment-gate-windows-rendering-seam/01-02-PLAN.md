<objective>
Prove GATE-01: an R session on macOS reports `tk windowingsystem == aqua` and can `load` a trivial Aqua-Tk C extension. This runs the deployment-gate spike (D-02) to establish that `aqua` is reachable at all, tests the D-01 package-local ideal (spike step 4) so the D-01-vs-D-02 decision is evidence-grounded, then commits a repeatable smoke test (D-03): a trivial Tk-linked extension + a scriptable assert.

</objective>
