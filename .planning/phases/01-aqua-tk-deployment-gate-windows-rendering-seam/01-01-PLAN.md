<objective>
Isolate all native window/context creation behind a 5-function platform seam (`gfx_backend.h`) and move the existing Win32 window/context code behind it unchanged (`gfx_backend_wgl.c`). This is a pure mechanical refactor (D-05): no rendering behavior changes.

Purpose: Confine platform code to a single backend translation unit so the tier-neutral GL draw pass no longer sees Win32 types — the prerequisite for the Phase 4 macOS backend behind the same seam.
Output: `gfx_backend.h` (opaque seam, D-04), `gfx_backend_wgl.c` (Win32 backend, `#ifdef _WIN32`), and the core refactored to call only `gfx_*`.
</objective>
